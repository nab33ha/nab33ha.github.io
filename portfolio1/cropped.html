Cropped Letters

One of the first projects I did as part of my Core Typography course, introduces letterforms as a fundamental element in communication design.

This project entailed developing a basic understanding of type anatomy and applying digital methods to learn about letters as part of the visual tapestry we call Design. The syncopated and repetitive nature of the forms creates a tiled composition which helps understand the perspective of simple design.
